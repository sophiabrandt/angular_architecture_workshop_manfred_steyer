Get your Starter Kit

Option A: Clone the Starterkit

Clone the starterkit and install the npm dependencies.

git clone https://github.com/manfredsteyer/2022_09_20_ENT.git
cd 2022_09_20_ENT
npm i

Option B: Fallback

If option A does not work, download and unzip the starter kit. If you are on Windows, don't use the Windows-internal Zipper but 7zip or winzip:

  • Starterkit

Mac/Linux: chmod -R +x node_modules
